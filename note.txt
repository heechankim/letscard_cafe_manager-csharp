요구사항

1. 미낙찰 다시 100원 경매로 돌릴 수 있어야한다.
2. 경매 종료후 뎃글이 0개가 달리면 미낙찰
3. 회원아이디는 뎃글에 써있는 아이디 기준으로 하자
4. 사진찍는 기준은 보내준 사람 기준
5. 단체싱글 판매자는 소수라 정해져 있음

6. 업로드 횟수, 뎃글 단 횟수 카운트가 있어야함

수수료 계산

입금해줄 금액 = 낙찰금 - 수수료 / seller_get = bid - fee
수수료 = 등록수수료 + 기본수수료 / fee = upload_fee + basic_fee
기본수수료 = 낙찰금 * 0.1 / basic_fee = bid * 0.1
